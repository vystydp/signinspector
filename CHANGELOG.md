### Added
- Initial release of SignInspector
- Client-side PDF signature validation
- SignInspectorViewer component for embedding in web apps
- Core validation functions (validatePdfBytes, validatePdfFile)
- PDF parsing module for extracting signatures
- CMS/PKCS#7 validation module
- Event system for viewer notifications
- Comprehensive TypeScript types
- Demo web application
- Unit tests with Vitest
- Integration tests with Playwright
- API documentation with TypeDoc
- README with usage examples
- MIT license
